## 🔧 技术实现

### 查询SQL
```sql
SELECT * FROM (
    SELECT 
        id, sn as user_sn, user_name, name, mobile, 
        address, sex, create_date, modify_date, 
        login_date, is_enabled, c_vip_id, cardno
    FROM CMS_USER
    WHERE is_enabled = 1
    ORDER BY create_date DESC, id DESC
) WHERE ROWNUM <= 10
```
